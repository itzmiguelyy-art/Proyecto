Fibonacci y Números Primos

Contexto del Problema

La serie de Fibonacci aparece en muchos patrones de crecimiento de la naturaleza y las matemáticas. Los números primos son utilizados en áreas importantes como la seguridad informática y la criptografía. Este proyecto combina ambos conceptos para generar números Fibonacci y detectar cuáles son números primos.

Explicación del Algoritmo

El usuario ingresa una cantidad de términos. JavaScript genera la secuencia Fibonacci utilizando ciclos y variables. Después, cada número se analiza para verificar si es primo. Finalmente, los resultados se muestran en pantalla.

Conclusión

Las matemáticas ayudan a resolver problemas reales mediante patrones y lógica. Fibonacci permite representar crecimiento y los números primos ayudan en sistemas de seguridad y validación.
